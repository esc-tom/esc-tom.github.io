PART 1: FIREBASE SETUP (10 minutes)
================================================================================

[ ] 1. Go to https://console.firebase.google.com/

[ ] 2. Click "Add project"
    - Project name: annotation-tool
    - Disable Google Analytics
    - Click "Create project"

[ ] 3. Add Web App
    - Click web icon (</>)
    - App nickname: Annotation Tool
    - Click "Register app"

    - Click "Continue to console"

[ ] 4. Enable Firestore Database
    - Click "Firestore Database" in left sidebar

    - Choose location: us-central (or closest to you)
    - Click "Enable"

[ ] 5. Set Firestore Security Rules
    - Click "Rules" tab

    - Click "Save"

PART 2: CODE UPDATES (5 minutes)
================================================================================

[ ] 7. Update index.html
    - Open: index.html
    - Find the closing </head> tag or before <script src="scripts/script.js">
    - Add these 3 lines:
      <script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app-compat.js"></script>
      <script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-firestore-compat.js"></script>
      <script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-auth-compat.js"></script>
    - Add this line after Firebase CDNs:
      <script src="scripts/firebase-storage.js"></script>

[ ] 8. Update script.js - Add Firebase config
    - Open: scripts/script.js
    - At the TOP of the file, add:
      
      // ========== FIREBASE CONFIGURATION ==========
      const FIREBASE_CONFIG = {
          apiKey: "YOUR-API-KEY-HERE",
          authDomain: "your-project.firebaseapp.com",
          projectId: "your-project-id",
          storageBucket: "your-project.appspot.com",
          messagingSenderId: "123456789",
          appId: "1:123456789:web:abc123"
      };

      let firebaseStorage = null;
      let firebaseReady = false;

      async function initFirebaseStorage() {
          if (firebaseReady) return true;
          firebaseStorage = window.firebaseStorage;
          const success = await firebaseStorage.init(FIREBASE_CONFIG);
          if (success) firebaseReady = true;
          return success;
      }
      // ========== END FIREBASE CONFIGURATION ==========

    - Replace the config values with YOUR Firebase config from step 3!

[ ] 9. Update script.js - Update DOMContentLoaded
    - Find: document.addEventListener('DOMContentLoaded', ...
    - Change to:
      document.addEventListener('DOMContentLoaded', async () => {
          await initFirebaseStorage();
          init();
      });
